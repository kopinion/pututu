*** 名词解释
| 缩写 | 全称                  | 说明                                                                           |
|------+-----------------------+--------------------------------------------------------------------------------|
| TAP  | Terminal Access Point | 经常是一个硬件设备，用来在局域网中监听网络事件，以便于管理员对网络进行分析。   |
|      |                       | 一般组成有三个部分：Port A  --- Monitor --- Port B, 在计算机网络中，通常指的是 |
|      |                       | 虚拟网络接口，TAP 经常工作在网络模型中链接层（第二层）                         |
| TUN  | network TUNnel        | 模拟了一个工作在网络层（network layer, 第三层）的设备，处理 IP 包              |
|      |                       |                                                                                |
